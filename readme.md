# Learning Path Recommender

A system designed to help users progress from their current skill level to a clearly defined learning goal within a specific domain (such as becoming a junior web developer or learning the basics of data analysis).

##  Features
- Takes inputs such as **current skills**, **background**, and **time availability**
- Generates a **prioritized learning path**
- Produces a **4–6 week personalized roadmap**
- Provides **resource suggestions** (videos, articles, websites)
- Ensures a **focused and achievable** learning journey

##  Files in This Project
- **problemstatement.md** – Detailed description of the problem  
- **usecases.md** – Key scenarios where the system will be used  
- **readme.md** – Project overview and instructions  

##  How It Works
1. User enters their current skills and learning goal  
2. System analyzes input and identifies gaps  
3. A structured learning path is generated  
4. A weekly action plan (4–6 weeks) is provided  
5. User follows roadmap and tracks progress  

##  Goal of the System
To offer a realistic, guided, and efficient learning experience by avoiding overwhelming information and giving the user clear next steps.
