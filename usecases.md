# Use Cases – Learning Path Recommender

## 1. New Learner Entering a Field
**Description:**  
A user with little or no prior experience wants to start learning a domain (e.g., web development, data analysis).

**System Actions:**  
- Collect background and available time  
- Recommend a beginner-friendly learning path  
- Generate a 4–6 week starter roadmap  

---

## 2. User with Some Experience Seeking Structured Guidance
**Description:**  
A user who knows a few concepts but lacks a clear roadmap.

**System Actions:**  
- Identify skills they already have  
- Highlight missing prerequisite topics  
- Provide a prioritized, customized learning path  

---

## 3. Professional Upskilling for Career Growth
**Description:**  
A working professional needs targeted learning to meet career goals (e.g., transition to junior data analyst).

**System Actions:**  
- Analyze user’s current role and required target skills  
- Recommend learning modules aligned with industry needs  
- Map a realistic weekly plan based on time constraints  

---

## 4. Student Preparing for Projects or Internships
**Description:**  
A student looking to quickly build specific skills for an upcoming project or internship opportunity.

**System Actions:**  
- Identify skill gaps related to the project requirement  
- Provide a fast-track learning path  
- Suggest project ideas and resources  

---

## 5. User Wanting to Learn for Personal Interest
**Description:**  
A user interested in learning casually without professional pressure (e.g., learning basic Python for hobby).

**System Actions:**  
- Recommend lightweight, enjoyable resources  
- Create a low-pressure roadmap  
- Allow flexible time-based adjustments  

---

## 6. User with Limited Time and Needs Efficient Learning
**Description:**  
Someone with only a few hours per week but still wants progress.

**System Actions:**  
- Optimize topics to essentials only  
- Recommend short, effective resources  
- Provide a minimalistic weekly plan  

---

## 7. User Tracking Progress Over Time
**Description:**  
A returning user wants to continue learning where they left off.

**System Actions:**  
- Recall previously completed modules  
- Update learning path dynamically  
- Adjust upcoming roadmap based on progress
